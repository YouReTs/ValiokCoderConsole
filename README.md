Основные компоненты Lazarus, которые нужно добавить на форму:
TMainForm - главная форма

TImage (2 штуки) - для preview и source изображений

TComboBox (4 штуки) - для выбора категорий
TButton (4 штуки) - кнопки действий
TMemo - для лога
TGroupBox - для группировки
TLabel (несколько) - для текстовых меток
TOpenDialog - диалог открытия файлов
TSaveDialog - диалог сохранения файлов

Установка необходимых компонентов:
Для работы с изображениями в Lazarus рекомендуется установить пакет BGRABitmap:

В меню Lazarus: Package > Install/Uninstall Packages

Особенности реализации:
Архитектура: Приложение разделено на логические модули

Обработка изображений: Используется BGRABitmap для качественного масштабирования

Интерфейс: Реализовано переключение между компактным и расширенным видом


Логирование: Все действия записываются в Memo
Обработка ошибок: Добавлена обработка исключений
